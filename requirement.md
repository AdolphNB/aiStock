# 股票助手软件需求文档
    这是一款股票助手软件，用于帮助用户获取实时或者历史股票数据，以及借助LLM进行股票分析，提供股票交易策略，并且提供按策略进行股票监控。

## 软件简介
    - 软件名称：股票助手
    - 软件版本：1.0.0
    - 软件作者：张强
    - 软件邮箱：304633698@qq.com

## 环境要求
    - 环境要求：
        - 操作系统：Windows 10 或以上版本
        - 内存：8GB 或以上
        - 硬盘空间：20GB 或以上
        - 显卡：NVIDIA GTX 1060 或以上
    
    - 开发需求
        - 开发语言：Python 3.11 或以上版本
        - 开发环境：Visual Studio Code
        - 数据库：MySQL 或 SQLite
        - UI框架：PyQt6
        - 版本管理：Git
        - 使用uv进行环境管理

## 约束条件
    - 软件的架构要极其清晰，模块之间的依赖要最小化，每个模块的功能要单一。
    - 软件的性能要满足用户的需求，即能够在短时间内完成股票数据的获取和分析。
    - 软件的稳定性要高，能够在长时间运行下保持正常工作。
    - 软件的用户界面要友好，操作简单。
    - 软件的代码要符合Python的编码规范，注释要详细。
    - 软件的数据库要安全，不能泄露用户的个人信息。
    - 代码的扩展性要非常好，能够方便地添加新的功能模块。
